For my front end, I use the Angular Framework. Angular allows me to manage user interactions, create and send requests to my service, and validate forms.

I create my services are built using .NET Core 3. For architecture, I use .NET Core MVC and RESTful patterns. MVC allows you to leverage many features such as; model validation, Cross-Site Request Forgery Protection, and can add documentation easily using Swagger.

My data access layer is built using Entity Framework Core (EF Core). For my actual database, depending on my developing environment, I either utilize MSSQL on Windows, or Sqlite on OSX. EF Core provides model validation against your database, type checking available in queries, removes the need mantaining any magic SQL strings, and EF migrations to easily keep track and update changes.
